2. Create `src/components/sections/Navbar.tsx` as a 'use client' component using HeroUI's Navbar:

```
import { Navbar, NavbarBrand, NavbarContent, NavbarItem, NavbarMenuToggle, NavbarMenu, NavbarMenuItem } from "@heroui/navbar";
import { Button } from "@heroui/button";
```

Structure:
- `NavbarBrand`: Logo image (next/image, ~140x36px) linking to top of page
- `NavbarContent` (center, hidden on mobile): Three NavbarItems with links — "How It Works" (#how-it-works), "Features" (#features), "Pricing" (#pricing)
- `NavbarContent` (right): "Login" as ghost Button (variant="light", className with pill radius), "Book a Demo" as primary Button (bg-primary text-white, pill radius, shadow-brand-sm, hover shadow-brand-md)
- `NavbarMenuToggle` for mobile hamburger (sm:hidden)
- `NavbarMenu` with NavbarMenuItems for mobile slide-out (same 3 links + 2 CTAs stacked)

Navbar props:
- `shouldHideOnScroll={false}` (always visible)
- `isBlurred={true}` for backdrop blur
- `maxWidth="xl"` to match 1280px content width
- `className="bg-bg-primary/80 backdrop-blur-lg border-b border-border/50"` for the semi-transparent sticky effect

Smooth scroll: Each nav link uses `onClick` handler that calls `document.querySelector(href)?.scrollIntoView({ behavior: 'smooth' })` and closes mobile menu. Do NOT use native anchor behavior (it jumps without smooth scroll).

Mobile menu state: Use React useState for `isMenuOpen`, pass to Navbar's `isMenuOpen` prop and `onMenuOpenChange`.

Typography: Nav links use font-body (Inter), text-sm, font-medium, text-text-secondary, hover:text-primary transition.
